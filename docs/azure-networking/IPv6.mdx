## Demo Scenarios
This is a demo for simple IIS website inside 2 VMs in an AvSet with IPv4/v6 dual-stack vNet , published with SLB onto IPv4/v6 Internet-facing PIP.   
![load-balancer-ipv6](https://docs.azure.cn/zh-cn/load-balancer/media/load-balancer-ipv6-overview/load-balancer-ipv6.png)

This demo also build with ARM templates, the build-in IaC capability in Azure.

Yes, this demo can be hosted in Global Azure or China Azure.
